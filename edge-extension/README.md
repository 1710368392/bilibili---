# Bilibili 用户备注助手

为 Bilibili 用户添加自定义备注，支持多标签和中国风配色。

## 功能

- 按住 Shift + 右键用户名即可添加备注
- 支持多个彩色标签
- 支持文本备注
- 数据跨页面同步
- 支持导出/导入备份

## 安装

1. 打开 Edge 浏览器，访问 `edge://extensions/`
2. 开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择本文件夹

## 使用

- 按住 **Shift** 键，**右键**点击任意用户名
- 在弹出的面板中添加标签和备注
- 点击扩展图标可管理所有备注
